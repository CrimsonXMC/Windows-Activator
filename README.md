# Windows-Activator
Activates Windows

Currently supports:
Windows 10/11 Pro
Windows 10/11 Pro N
Windows 10/11 Pro For Workstations
Windows 10/11 Enterprise 
Windows 10/11 Enterprise N
Windows 10/11 Home
Windows 10/11 Home N
For Windows 11, just select the Windows 10 key for your edition of windows (Pro, Enterprise, etc.)
